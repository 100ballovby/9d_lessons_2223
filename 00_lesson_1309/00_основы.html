Первый вид списка - <ul> - unordered list (маркированный список):

  • Первый
  • Второй
  • Третий

Второй вид списка - <ol> - ordered list (нумерованный список). У нумерованных списков есть два атрибута - start (отвечает за число, с которого начнется нумерация), type (отвечает за стиль элементов списка)

  1. i - маленькие римские цифры (iv, v, i)
  2. I - большие римские цифры (V, VIII, IX)
  3. a - маленькие буквы
  4. A - большие буквы
  5. 1 - числа (писать не нужно)
